# 📚 Plataforma Colegio

Este proyecto es una aplicación web para registrar asistencia de estudiantes usando Python y Flask.

## 🚀 Funcionalidades

* Registro de asistencia
* Lectura de datos desde Excel
* Reproducción de sonidos (ok / error)

## 📁 Estructura del proyecto

* `app.py`: archivo principal
* `asistencia/`: módulo de asistencia
* `templates/`: páginas HTML
* `static/`: archivos estáticos

## 🛠️ Tecnologías usadas

* Python
* Flask
* Excel (openpyxl o pandas)

## ▶️ Cómo ejecutar

1. Instalar dependencias:

   pip install -r requirements.txt
 
2. Ejecutar:

   python app.py

## 🌐 Despliegue

Este proyecto puede desplegarse en Render.

Desarrollado para proyecto educativo.
